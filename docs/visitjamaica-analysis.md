### Notes sur les URLs d'images Cloudinary (SimpleView)
Les images utilisent le CDN Cloudinary via SimpleView. Les parametres d'URL permettent des transformations:
- `c_fill` : Remplissage avec recadrage
- `c_limit` : Limite sans recadrage
- `h_XXX` : Hauteur en pixels
- `w_XXX` : Largeur en pixels
- `q_XX` : Qualite (75 = haute qualite)

Pour obtenir l'image en pleine resolution, utiliser l'URL sans parametres de transformation ou avec des dimensions plus grandes.
